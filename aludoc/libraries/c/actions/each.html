each

Iterates over an iterable, invoking its body for each element. The body is invoked in its own scope; the current element and information about the current loop will be available in that scope.

Summary

Name
each
Dynamic parameters
No

Parameters

NameTypeRequiredDescription
elementsIterableYesThe elements to iterate over.
element nameStringNoThe variable name to use for the current element; defaults to 'element'.
information nameStringNoThe variable name to use for information about the current loop; 'loop' by default.
